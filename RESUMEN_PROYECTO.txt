🚀 INSTALACIÓN RÁPIDA (3 OPCIONES)
═════════════════════════════════════════════════════════════════════════════

OPCIÓN 1: Docker (Recomendado)
────────────────────────────────
$ cd kdp-author-manager
$ chmod +x install.sh
$ ./install.sh

✓ Elige "s" para Docker
✓ Espera a que termine (3-5 minutos)
✓ Accede a http://localhost:8000/admin

Email:    admin@kdpmanager.local
Password: password


OPCIÓN 2: Instalación Local
──────────────────────────────
$ cd kdp-author-manager
$ chmod +x install.sh
$ ./install.sh

✓ Elige "n" para instalación local
✓ Sigue las instrucciones
✓ Accede a http://localhost:8000/admin


OPCIÓN 3: Manual (Expertos)
────────────────────────────
$ composer create-project laravel/laravel kdp-author-manager
$ cd kdp-author-manager
$ cp .env.example .env
$ composer require filament/filament
$ php artisan filament:install --panels
$ php artisan migrate --seed
$ npm install && npm run build
$ php artisan serve
